## Product design system

Build a polished B2B SaaS app, not a landing page or AI demo.

Use:
- Next.js + Tailwind + shadcn/ui components
- neutral base colours
- CSS variables for theme tokens
- lucide-react icons
- excellent spacing and typography

Visual target:
- Supabase: clean developer tool, muted green accent, dark/light neutral UI
- ChatGPT: minimal, calm, spacious, very restrained
- Claude: warm, editorial, soft beige/stone palette

Default style:
- background: bg-background or bg-slate-50
- surfaces: bg-card / bg-white
- borders: border-border / border-slate-200
- text: text-foreground, text-muted-foreground
- radius: rounded-xl or rounded-2xl
- shadows: shadow-sm only
- spacing: generous, never cramped

Avoid:
- gradients
- glassmorphism
- neon colours
- decorative blobs
- emoji-heavy UI
- random purple/blue AI demo styling
- overly large hero sections inside the app
